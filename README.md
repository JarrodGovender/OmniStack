# OmniStack

Enterprise-grade custom data dashboards, AI automation, and secure software
development for South African SMEs.

## Status

Under active development. This repository currently hosts the foundation
of the OmniStack web platform, built on Streamlit.

## Local development

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

The app will open at http://localhost:8501
